# 这里只有你和我

这是只给两个人看的星夜回忆网站。相爱的每一天都是一颗可以重新找到的星，写下的话也不需要像公开展示那样克制。

## 安全启动

现在必须通过带认证的本地服务启动，不能再直接双击 `index.html`：

```bash
python3 server.py
```

然后打开 `http://127.0.0.1:4173`，输入约定的 8 位密码。

安全层包含：

- PBKDF2 密码哈希与恒定时间比较，源码中不保存明文密码
- 浏览器会话级、`HttpOnly`、`SameSite=Strict` 的签名 Cookie；关闭浏览器后需重新输入密码，服务端有效期最长 12 小时
- 登录失败限速：10 分钟内连续失败 6 次会锁定 15 分钟
- 同源请求检查、JSON 请求体大小限制、严格输入白名单
- CSP、禁止 iframe、禁止 MIME 嗅探、无引用来源等安全响应头
- 私密页面、图片、星图数据和脚本在未登录时均不能直接读取

项目没有使用 SQL 数据库，心情、留言、愿望与回答使用经过严格校验的 JSON 文件保存，因此不存在把输入拼接进 SQL 的路径。用户输入在前端只通过 `textContent` 渲染，避免注入 HTML/脚本。

> 原来的纯静态托管（例如 GitHub Pages）无法真正保护源文件。公开上线时需把 `server.py` 部署到支持 Python 常驻服务和持久化磁盘的平台，并使用 HTTPS。

在 HTTPS 反向代理后运行时，将环境变量 `STAR_TREE_SECURE_COOKIE=1`，会为会话 Cookie 增加 `Secure` 属性。

## 当前包含

- 电影级横版与竖版主视觉，桌面和手机按设备选择资源
- 「今夜」：当天想说的话和一段随机从前
- 「星图」：按年份、月份和日期浏览一起走过的夜
- 「我们」：时间统计、纪念章节和喜欢的日常细节
- 「此刻」：两个人的心情轨道、共享星空留言、未来时光胶囊和今晚一问
- 「给你」：私密长信、可勾选和添加的愿望清单、周年倒计时
- 心情、留言、愿望和今晚回答会自动同步；页面停留时约每 12 秒刷新一次
- 本机记住当前是 Yuya 还是 Zhennan，填写心情、留言和回答时自动沿用
- 页面右上角可以主动退出私密会话
- 日期直达、随机回忆和可分享的 `?day=` 链接
- 分镜式入场、流星、呼吸轨道、指针光晕、视图分层转场
- 自动低功耗模式、减少动态效果支持、静态星图渲染

页面不需要安装第三方依赖，Python 3 即可运行。

## 修改开始日期

在 `script.js` 顶部修改：

```js
relationshipStartDate: "2024-09-09",
```

## 填写星星内容

只需要编辑 `data/stars.js`：

```js
{
  day: 100,
  title: "标题",
  message: "这一天想留下的话。",
  type: "memory"
}
```

`type` 可选：

- `normal`：普通日子
- `memory`：特别回忆
- `anniversary`：纪念日
- `letter`：长信
- `future`：未来信件

没有填写内容的日期仍然可以从星图打开，并会显示温和的默认文案。

## 性能策略

- 首屏背景约 360 KB，横版和竖版只加载当前需要的一张
- 今夜场景最多绘制 42 颗轻量动态星，手机进一步减少
- 动画限制在约 25 FPS，流星复用同一个 Canvas，并在后台、弹层或非今夜视图停止
- 星图按当前年份静态绘制，不持续刷新
- Canvas 像素比上限为 1.5，低功耗设备自动降为 1
- 指针视差只在精确指针和非低功耗设备上启用
- 尊重系统的 `prefers-reduced-motion` 设置
